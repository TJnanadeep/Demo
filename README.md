A beautiful, responsive weather application that provides detailed weather information with visual elements.

### Features

- **Current Weather Data**: Real-time weather information for any location
- **Detailed Metrics**: Temperature, humidity, wind speed, pressure, visibility, and more
- **5-Day Forecast**: Visual forecast cards for the next 5 days
- **Location Support**: Search by city name or use your current location
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Visual Elements**: Beautiful gradients, weather icons, and smooth animations

### How to Use

1. **Open the Weather App**:
   Simply open `weather_app.html` in any modern web browser.

2. **Search for a City**:
   - Type a city name in the search box
   - Click the "Search" button or press Enter
   - View detailed weather information

3. **Use Current Location**:
   - Click the üìç location button
   - Allow location access when prompted
   - View weather for your current location

### Files

- `weather_app.html` - Main HTML structure
- `weather_style.css` - Styling and responsive design
- `weather_script.js` - Weather data fetching and display logic

### Weather Data Includes

- Current temperature and "feels like" temperature
- Weather conditions with icons
- Wind speed and direction
- Humidity levels
- Atmospheric pressure
- Visibility
- Cloudiness percentage
- Sunrise and sunset times
- 5-day forecast with daily temperatures

### Technical Details

- Built with vanilla HTML, CSS, and JavaScript (no frameworks required)
- Uses OpenWeatherMap API for weather data
- Fully responsive design with CSS Grid and Flexbox
- Smooth animations and transitions
- Error handling for invalid cities or API failures

### Requirements

- Any modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection to fetch weather data
- Location services (optional, for current location feature)

## Sample Python Script

This repository also contains a sample Python script (`sample_script.py`) that demonstrates various Python programming concepts.

### Features

The script demonstrates:
- **Object-Oriented Programming**: A `Person` class with methods
- **Functions**: Various utility functions for calculations and data processing
- **Error Handling**: Try-except blocks for robust error management
- **File Operations**: Reading and writing JSON files
- **Data Structures**: Working with lists and dictionaries

### Usage

Run the script using Python 3:

```bash
python3 sample_script.py
```

Or make it executable and run directly:

```bash
chmod +x sample_script.py
./sample_script.py
```

### What the Script Does

1. Creates and demonstrates Person objects
2. Performs mathematical calculations (sum and average)
3. Processes data structures
4. Demonstrates error handling
5. Saves and reads data from JSON files

### Requirements

- Python 3.6 or higher
- No external dependencies (uses only standard library)
